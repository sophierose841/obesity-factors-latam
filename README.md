# Examining Factors Contributing to Overweight and Obesity Rates in Mexico, Colombia and Peru 
## About
Using both an unsupervised and supervised learning method as part of a university assignment. The purpose of this project was to understand how to prepare and organise data before implimenting both unsupervised machine learning methods and supervised methods.

## Introduction
The purpose of this assignment was to experiment using unsupervised and supervised methods to learn the key skills of data preprocessing, visualisation, model implimentation and evaluation methods. Particularly becoming familiar with adjusting model parameters and evaluating the best choice for the final model.
